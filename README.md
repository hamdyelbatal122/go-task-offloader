# Material Design — Register & Login Form

A clean, modern **Register & Login Form UI** inspired by Google's Material Design principles. Built with pure HTML and CSS — no frameworks required.

## ✨ Features

- 🎨 Material Design aesthetic (flat, card-based layout)
- 📋 Login and Registration forms
- ✅ Input field focus animations
- 📱 Fully responsive on all screen sizes
- ⚡ Zero JavaScript dependencies

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/hamdyelbatal122/Login-form.git
   cd Login-form
   ```

2. **Open in browser**
   ```bash
   open index.html
   ```

## 📁 Project Structure

```
Login-form/
├── index.html        # Login form
├── register.html     # Registration form
├── style.css         # Material Design styles
└── README.md
```

## 🎨 Customization

Update the CSS variables to match your brand colors:

```css
:root {
  --primary-color: #4A90E2;
  --accent-color:  #E91E63;
}
```
